Get Started with Installing ChimeraX Bundles

Is ChimeraX installed? If not, download it from here: https://www.rbvi.ucsf.edu/chimerax.html

Is ChimeraX running? If not, go ahead and start it.

After ChimeraX has started, click the Tools menu, select the General category and click on Toolshed.

Navigate to the bundle you want. Now click Download.

Wait for it...

Okay! Your bundle is now installed. Look in the Tools menu for newly installed tools.
